UI Starter Storybook

1. From within the root of the project run the command `npm install` to install the dependencies 
2. Run the following command `npm run storybook` to start the webpack dev server. Once started it should open a browser (localhost:6006 or the next available port).
3. You can now modify CSS, JS, SCSS, and TS files and see the changes immediately reflected in the webpack dev server.
4. Run `npm run build:htl` to Convert Handlebar files to HTL
